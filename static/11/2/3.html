Уравнение сферы

Прежде чем вывести уравнение сферы введем понятие уравнения поверхности в пространстве. Для этого рассмотрим прямоугольную систему координат Oxyz и некоторую поверхность F. Уравнение с тремя переменными x, y, z называется уравнением поверхности F, если этому уравнению удовлетворяют координаты любой точки поверхности F и не удовлетворяют координаты никакой другой точки. Пусть сфера имеет центром точку C ( x 0 ; y 0 ; z 0 ) и радиус R. Расстояние от любой точки M ( x ; y ; z ) до точки C вычисляется по формуле:

M C = ( x – x 0 ) 2 + ( y – y 0 ) 2 + ( z – z 0 ) 2

Исходя из понятия уравнения поверхности, следует, что если точка М лежит на данной сфере, то M C = R , или M C 2 = R 2 , то есть координаты точки М удовлетворяют уравнению:

R 2 = ( x – x 0 ) 2 + ( y – y 0 ) 2 + ( z – z 0 ) 2
